# ws-test-server

Server for Test Task from wemake.services

Handles requests from the [client](https://github.com/standbyoneself/ws-test-client) to display GitHub user repositories

Powered by Express, TypeScript, Octokit

## Transpile

```
tsc
```

### Run the server

```
yarn start
```
